# causal-descent

Replication code for _Double and Single Descent in Causal Inference
with an Application to High-Dimensional Synthetic Control_ by Jann Spiess, Guido Imbens, and Amar Venugopal (https://arxiv.org/abs/2305.00700).

Data for the LaLonde (1986) wage study made available by Dehejia & Wahba (1999) and hosted by [NBER](https://users.nber.org/~rdehejia/data/.nswdata2.html). Data for the Abadie et al. (2010) California smoking study sourced from the [CDC](https://chronicdata.cdc.gov/Policy/The-Tax-Burden-on-Tobacco-1970-2019/7nwe-3aj9).
